### 为什么选这个？

- 🌙 今天是工作日，时间不早了（22:00）
- 💤 御主的目标之一是"养成早睡习惯"
- ⚡ 快速实现，立即可用（1 小时内完成）
- 💕 温柔提醒，不强迫

### 核心功能

1. **⏰ 就寝时间设置** - 设置目标就寝时间（默认 23:00）
2. **⏱️ 倒计时提醒** - 实时显示距离就寝还有多久
3. **💡 睡前小贴士** - 5 条科学睡眠建议
4. **🌙 温暖氛围** - 深蓝色渐变背景 + 星星动画

### 技术栈

- **前端**：HTML + CSS + JavaScript（纯前端，无需后端）
- **设计**：温暖柔和的夜晚氛围
- **存储**：localStorage（记住用户设置）
- **服务器**：Python HTTP Server（端口 8083）

### 设计理念

- **温暖柔和** - 深蓝色渐变背景，营造夜晚氛围
- **简洁直观** - 一眼就能看懂的操作界面
- **鼓励而非强迫** - 用温柔的提醒代替刺耳的闹钟
- **仪式感** - 睡前放松仪式，帮助入睡
