# 晚安小助手 - 早睡提醒 Web App

**创建时间**: 2026-02-26 22:00  
**目的**: 帮助御主养成早睡习惯  
**技术栈**: HTML + CSS + JavaScript（纯前端）  
**特点**: 无需后端，开箱即用

---

## 🎯 核心功能

1. **就寝时间设置** - 设置目标就寝时间
2. **倒计时提醒** - 距离就寝还有多久
3. **睡前仪式** - 放松音乐 + 冥想引导
4. **睡眠小贴士** - 科学睡眠建议
5. **睡眠记录** - 记录实际入睡时间

---

## 🌟 使用方式

```bash
# 启动服务器
python3 -m http.server 8083

# 访问
http://101.132.81.50:8083/
```

---

## 🎨 设计理念

- **温暖柔和** - 深蓝色渐变背景，营造夜晚氛围
- **简洁直观** - 一眼就能看懂的操作界面
- **鼓励而非强迫** - 用温柔的提醒代替刺耳的闹钟
- **仪式感** - 睡前放松仪式，帮助入睡

---

## 💡 睡眠小知识

1. **固定作息** - 每天同一时间入睡和起床
2. **避免蓝光** - 睡前1小时不看手机
3. **放松仪式** - 热水澡、冥想、阅读
4. **温度控制** - 卧室温度保持在18-22°C
5. **黑暗环境** - 使用遮光窗帘
