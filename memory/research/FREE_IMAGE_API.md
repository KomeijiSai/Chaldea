## 💡 优化建议

### 1. 按需生成
- 只在御主要求时生成
- 不自动生成
- 减少无效消耗

### 2. 批量生成
- 一次生成多张
- 选择最好的保存
- 减少API调用次数

### 3. 场景分级
- **P0 场景**（御主特别要求）: 用高质量 API
- **P1 场景**（日常打卡）: 用免费 API
- **P2 场景**（批量测试）: 用最便宜的 API

### 4. 缓存复用
- 相似场景复用已有照片
- 建立"基础姿势"库
- 只生成全新场景
