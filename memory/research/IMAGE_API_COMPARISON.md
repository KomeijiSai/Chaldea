## ⚠️ 注意事项

### 1. **代理问题**
- Replicate、OpenAI、Stability AI、Leonardo.ai、RunwayML、Midjourney 都需要代理
- 阿里云百炼、火山引擎国内可直接访问

### 2. **质量 vs 成本**
- Flux-Schnell: 质量低，但极便宜
- DALL-E/阿里云百炼: 质量高，价格适中
- Midjourney: 质量最高，但订阅制成本高

### 3. **稳定性**
- 大平台（OpenAI、阿里云）更稳定
- 小平台可能有 API 变更风险
