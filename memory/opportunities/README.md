## 商机评估标准

| 维度 | 权重 | 说明 |
|------|------|------|
| 变现潜力 | 30% | 能挣多少钱 |
| 投入成本 | 25% | 时间、精力、资金 |
| 御主优势 | 25% | 是否匹配御主的技能和资源 |

| 可持续性 | 10% | 能否长期做 |
